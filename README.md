# Callum Boaden Web Design & Development Portfolio

Premium, high-performance freelance portfolio and agency website built for **Callum Boaden Web Design & Development** (hosted at `callumboaden.com.au`).

## Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4 (Dark-mode first design)
- **Icons**: Lucide React / Custom Inline SVGs
- **Animations**: Framer Motion 12 (Subtle micro-interactions)
- **Accessibility**: WCAG AA compliant semantic markup

## Getting Started

First, install dependencies:
```bash
npm install
```

Second, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Deploy automatically by pushing to the `main` branch.
